# Nextgen Cloud Engineering Course Assignments

Welcome to the central repository for my Cloud Engineering assignments!

## Directory Structure

This repository contains multiple assignments. Below is an overview of the current projects:

### 1. [Kubernetes Lab 1: PHP + MySQL](./Kubernetes/)
**Path:** `/Kubernetes/`

This assignment focuses on containerizing and deploying a stateful PHP web application (a Student Portal) using Docker and Kubernetes. 

**Key Objectives Completed:**
- Dockerized a PHP/Apache web application.
- Containerized a MySQL database and initialized it with a custom schema.
- Migrated the setup to **Kubernetes**.
- Deployed MySQL with Persistent Volume Claims (PVCs) for data retention.
- Configured Secrets and ConfigMaps for secure environment variables.
- Deployed the PHP web application alongside `phpMyAdmin` using Kubernetes Deployments and Services.

To view the Kubernetes YAML manifests and documentation for this specific lab, navigate to the [Kubernetes directory](./Kubernetes/).
